# Boondock Tango

**Boondock Tango** is here! We've decided to release a version you can buy and build at home—anywhere in the world. We firmly believe in the power of open-source goodness.

---

## About Boondock Tango

Boondock Tango is a **receive-only device for scanners and radios**

> **Note:** 
> - Currently works on Windows.
> - Mostly works on some flavors of Linux -- We're resolving some issues
> - macOS support is coming soon.
